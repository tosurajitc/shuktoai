# AI Services Website

A modern website for AI development services, featuring traditional AI, GenAI, and Agentic AI solutions.

## Project Structure

- `frontend/`: Next.js web application
- `backend/`: FastAPI server

## Development Setup

### Prerequisites
- Node.js (v14+)
- Python (v3.8+)
- Git

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
